Dialog

Корпоративный мессенджер для совместной работы и безопасного общения

Преимущества приложения Dialog

  • Эффективная командная работа

    Dialog позволяет настроить интеграции с сервисами компании и оптимизировать бизнес-процессы.

  • Безопасное общение

    Соответствие отраслевым стандартам безопасности, контроль и защита на разных уровнях.

  • Сокращение затрат на коммуникации

    Совмещение разрозненных инструментов в одной платформе оптимизирует расходы на ПО, оборудование, обслуживание и поддержку.

Функции Dialog

Групповые чаты и каналы

Группы — это чаты для общения, создайте их для команд и проектов, чтобы сфокусировать усилия, сделать работу более прозрачной, а информацию доступной для всех

Обмен файлами

Тяжелая презентация не отправляется по почте? Делитесь файлами, не задумываясь о лимитах по размеру или формату.

Аудио- / видеозвонки

Созванивайтесь с коллегам в мессенджере или подключайтесь к конференциям в АКС/ВКС-системе.

Поиск

Удобный поиск по чату поможет найти презентации, изображения, документы, а интеграция с адресной книгой — нужного сотрудника по имени, должности или отделу.

Чат-боты

Создавайте ботов и чат-ботов для автоматизации бизнес-процессов с помощью Bot SDK.

Защищенная коммуникация

  • Высокие стандарты безопасности

    Интеграция с инфраструктурой информационной безопасности

    • Интеграция с Active Directory AD / LDAP

    • DLP — предотвращение утечек

    • Интеграция с Антивирусным ПО

    • SIEM — идентификация угроз

    • Шифрование передачи данных по протоколам TLS, DTLS

    • Независимый аудит безопасности ПО

  • Данные под вашим контролем

    Вы можете выбрать предпочтительные варианты размещения Dialog

    • On-premise Размещение на серверах вашей компании

    • Private cloud Размещение на инфраструктуре стороннего провайдера в частном облаке

Dialog помогает командам стать эффективнее

Переключение между разными приложениями нарушает концентрацию сотрудников. Используя Bot SDK и Dialog API, вы можете создавать интеграции со сторонними приложениями и автоматизировать рутинные процессы.

Bot SDK

Bot SDK позволяет создавать ботов и чат-ботов для Dialog. Боты оптимизируют повторяющиеся рутинные операции и освобождают время для более сложных задач. SDK доступен для JS, Java и Python.

Реализовано с помощью Bot SDK

Помощник HR bot

Упрощает работу с документами: опрашивает сотрудника по ключевым вопросам и формирует заявление на отпуск.

Dialog API

API позволяет другим приложениям взаимодействовать и обмениваться данными с Dialog.

Реализовано с помощью Dialog API

Совместная работа над документами МойОфис

Возможность автоматически создавать группы для обсуждения, уведомлять о предоставлении доступа и отправлять ссылки на документы в Dialog, не выходя из офисных приложений МойОфис.

Dialog в большом бизнесе

Сбербанк запустил корпоративный мессенджер СберЧат на базе Dialog для более чем 300 000 сотрудников.

Оперативное взаимодействие и обмен данными важны для компаний любого размера и профиля деятельности. При этом банку и компаниям экосистемы было необходимо собственное масштабируемое решение, обеспечивающее эффективную и защищенную коммуникацию большому количеству сотрудников. Мы реализовали решение, которое позволит нам не только оптимизировать процесс коммуникации, но и повысить эффективность совместной работы за счет создания единого цифрового рабочего пространства.

Первый заместитель председателя правления Сбербанка
