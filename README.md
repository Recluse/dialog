# dialog
--
## Основные команды для работы:

* Установка
```
npm i
```
* Запуск локального сервера 
```
npm start
```
* Сборка проекта без запуска локального сервера
```
npm run build
```
* Запуск тестирования на соответствия код-гайдам
```
npm test
```
## Структура файлов:
 * Исходный код — директория `src` 
 
 * Cборка — директория `build`
